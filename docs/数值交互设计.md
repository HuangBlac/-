# 克苏鲁研究院 - 数值交互设计文档

## 概述

本文档详细描述游戏的两大核心数值系统：**科研线**与**克苏鲁线**的交互机制。

---

## 一、科研线数值系统

### 1.1 核心数值定义

| 数值 | 范围 | 说明 |
|------|------|------|
| **知识 (Knowledge)** | 0-∞ | 学术能力的核心体现，通过课程和阅读获取 |
| **灵感 (Inspiration)** | 0-∞ | 创新思维的来源，影响idea质量和实验成功率 |
| **声望 (Reputation)** | 0-∞ | 学术影响力，影响投稿成功率 |
| **技能 (Skills)** | 0-∞ | 专业能力，影响特定行动的效率和考试分数 |

### 1.2 课程学习系统

#### 1.2.1 课程分类

```
必修课（3门，固定）：
├── 拉莱亚语言初步  → +3 拉莱亚语言
├── 神话文本阅读    → +3 文本解读
└── 形式科学导论   → +3 形式科学

选修课（从4门中选择3门）：
├── 法术与召唤法阵     → +2 神秘生物学, +1 说服
├── 旧日支配者认知科学 → +2 神话学, +1 密码学
├── 超自然邪教与混血种族 → +2 田野调查, +1 说服
└── 神秘生物学         → +3 神秘生物学
```

#### 1.2.2 课程 → 研究方向 映射

课程选择决定**研二时的研究方向**：

| 选修课类型 | 对应研究方向 |
|------------|--------------|
| 法术/召唤相关 | 法术与超自然科技分析 (ARCANE_ANALYSIS) |
| 文本/邪教相关 | 神话文本与仪式构造 (MYTHOS_RITUAL) |
| 邪教/混血相关 | 神明附属种族与独立种族 (DEITY_RACE) |
| 旧日支配者相关 | 旧日支配者与外神 (OUTER_GOD) |

#### 1.2.3 课程对科研的影响路径

```
课程学习
    │
    ├── 直接影响：技能点提升
    │       │
    │       └──→ 考试分数 = 60 + random(-10,15) + 技能值×2 + 知识÷10
    │
    └── 间接影响：研究方向决定
            │
            └──→ 决定可获取的idea池
                    │
                    └──→ 决定可使用的实验方法
```

### 1.3 小论文 (Paper) 研究系统

#### 1.3.1 Idea 生命周期

```
┌─────────────────────────────────────────────────────────────────┐
│                        Idea 状态流转                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   阅读文献 ──→ 原始idea ──→ 评估 ──→ 初步想法 ──→ 实验 ──→ 成熟想法
│     (100%)           │          │              │           │
│                      │          ▼              │           │
│                      │      ┌───┴────┐         │           │
│                      │      │ 丢弃    │ 改进     │           │
│                      │      │ (+技能) │ (+技能) │           │
│                      │      └───┬────┘         │           │
│                      │          │              ▼           │
│                      │          │        ┌────────┐        │
│                      │          │        │实验失败│        │
│                      │          │        │ (-理智)│        │
│                      │          │        └────────┘        │
│                      │          │              │           │
│                      │          │        ┌────────┐        │
│                      │          │        │实验成功│        │
│                      │          │        │ (+结果)│        │
│                      │          │        └────────┘        │
│                      │          │              │           │
│                      │          │         ≥2个结果        │
│                      │          │              │           │
│                      │          │              ▼           │
│                      │          │        ┌─────────┐       │
│                      │          │        │成熟想法 │       │
│                      │          │        │(可投稿) │       │
│                      │          │        └─────────┘       │
│                      │          │              │           │
│                      └──────────┴──────────────┘           │
│                                  │                          │
│                                  ▼                          │
│                       3个成熟想法 ──→ 撰写初稿 ──→ 投稿    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 1.3.2 Idea 核心属性

| 属性 | 说明 |
|------|------|
| **name** | Idea名称 |
| **description** | 描述 |
| **innovation** | 创新值 (1-10) |
| **direction** | 对应的研究方向 |
| **status** | 状态 (原始/初步/成熟) |
| **experiment_results** | 实验结果列表 |
| **progress** | 评估进度 |

#### 1.3.3 实验系统

**实验成功概率**：
```
成功率 = 50% + 知识×2% + 灵感×1% + 导师好感×0.1%
```

**实验方法**（按研究方向）：

| 研究方向 | 可用实验方法 |
|----------|--------------|
| 法术与超自然科技分析 | 形式科学建模分析、计算机模拟、数据收集与统计 |
| 神话文本与仪式构造 | 钻研典籍、文本比对、考古验证 |
| 神明附属种族与独立种族 | 田野调查、DNA分析、社会学研究 |
| 旧日支配者与外神 | 理论研究、数学建模、哲学思辨 |

#### 1.3.4 论文质量计算

```
论文质量 = 创新值平均×2 + 实验数量×2 + 契合性(同方向+10) + 多样性(≤10)
```

**投稿成功率**：
```
基础成功率 = 30%
成功率 += 论文质量×0.4%
成功率 += 已发表论文数×5%
成功率 += 声望×0.2%
```

**投稿结果分布**：
- 10%：Desk Reject（论文销毁）
- 27%：Major Reject（大修，进度-40%，损失2个实验结果）
- 18%：Minor Reject（小修，进度-10~30%）
- 45%：Accept（发表成功）

### 1.4 学位论文 (Thesis) 系统

#### 1.4.1 与小论文的差异

| 维度 | 小论文 | 学位论文 |
|------|--------|----------|
| 触发条件 | 研二开始，3个成熟idea | 研三，有≥1篇已发表小论文 |
| 完成要求 | 3个成熟idea | 开题→初稿→盲审→答辩 |
| 创新要求 | 创新值平均 | 需要**创新点成熟度**达标 |
| 失败惩罚 | 论文作废，可重写 | 可能延期毕业 |
| 理智消耗 | 较少 | 较大（需要整合多年研究） |

#### 1.4.2 学位论文阶段

```
学位论文阶段：
├── 未开始（研三且有小论文时可选）
├── 开题报告
├── 初稿攥写
├── 盲审（需要创新点）
└── 答辩
```

## 二、克苏鲁线数值系统

### 2.1 理智 (Sanity) 系统

#### 2.1.1 理智值定义

| 理智值 | 等级描述 | 行为影响 |
|--------|----------|----------|
| 80-100 | 神志清醒 | 正常科研，无特殊效果 |
| 60-79 | 有些不安 | 轻微负面效果 |
| 40-59 | 精神恍惚 | 科研效率下降，可能触发幻觉 |
| 20-39 | 濒临崩溃 | 重大负面影响，强制触发里世界事件 |
| 0-19 | 彻底疯狂 | 游戏结束 |

#### 2.1.2 理智变化来源

**减少理智的行动**：

| 行动 | 理智损失 |
|------|----------|
| 上课 | -1~5 (随机) |
| 实验失败 | -1~3 |
| 调查 | -3~8 |
| 阅读禁忌文本 | -10~20 |
| 投稿被拒 | -3~8 |
| 里世界课程 | -15~25 |

**恢复理智的行动**：

| 行动 | 理智恢复 |
|------|----------|
| 休息 | +10~20 |
| 论文发表成功 | +5~10 |

---

### 2.2 理智 ↔ 科研 交互机制

#### 2.2.1 理智对科研的直接影响

```
理智值区间    │ 实验成功率修正   │ Idea质量修正   │ 论文可靠性
─────────────┼─────────────────┼───────────────┼─────────────
80-100 (正常) │      0%        │      0%       │   100%
60-79 (不安)  │     -5%        │     +10%      │   100%
40-59 (恍惚)  │     -15%       │     +25%      │   80%
20-39 (崩溃)  │     -30%       │     +40%      │   50%
<20 (疯狂)    │   不可控       │     +50%      │   20%
```

**设计理念**：
- 理智越低，越容易发现"不正常"的结论
- 但理智过低时，实验变得不可靠
- 形成风险-收益权衡

#### 2.2.2 理智对Idea生成的特殊影响

**低理智时可能生成特殊Idea**：
- 理智 < 40：可能获得"疯狂洞察"类型的idea
  - 创新值极高 (8-10)
  - 但实验结果可靠性降低

---

### 2.3 随机事件系统

#### 2.3.1 事件触发条件

```
事件触发 = 30%概率 (每次行动后检查)
```

#### 2.3.2 事件类型与理智关联

| 事件类型 | 触发条件 | 效果特征 |
|----------|----------|----------|
| 普通克苏鲁事件 | 无 | 理智-5~20, 知识/灵感/声望变化 |
| 里世界事件 | 理智<40 | 高收益高风险，可能获得禁忌知识 |
| 导师暗示事件 | 研二+ | 警告玩家注意理智 |
| 疯狂幻觉事件 | 理智<30 | 理智大幅损失，可能获得灵感 |

#### 2.3.3 里世界事件示例

```
【里世界教室】
触发条件：理智<40，上课行动

效果：
├── 获得大量技能点 (+5~10)
├── 获得禁忌知识 (+5~10知识)
├── 灵感大幅提升 (+5~15)
└── 理智大幅损失 (-15~25)

特殊：此事件中获得的部分知识称为"污染知识"
      使用"污染知识"进行实验时，结果可靠性额外-20%
```

---

### 2.4 灵感 (Inspiration) 系统

#### 2.4.1 灵感获取途径

| 行动 | 灵感获取 |
|------|----------|
| 调查 | +3~8 |
| 阅读禁忌文本 | +5~10 |
| 随机事件 | +1~5 |
| 上课 | +0~2 |

#### 2.4.2 灵感对科研的影响

```
灵感影响：
├── Idea创新值加成：灵感÷10
├── 实验成功率加成：灵感×1%
└── 特殊：灵感极高(>50)时，可能触发"顿悟"事件
        顿悟：直接获得一个成熟idea（需消耗20灵感）
```

## 四、设计总结

### 4.1 核心权衡设计

1. **知识广度 vs 深度**：
   - 选修多样化课程 → 广度，但方向可能不明确
   - 专注特定方向 → 深度，但科研面窄

2. **理智 vs 科研成果**：
   - 高理智：实验可靠，但难有突破性发现
   - 低理智：容易发现异常，但结果不可靠

3. **安全 vs 风险**：
   - 正常上课/科研：稳定但平庸
   - 调查/阅读禁忌：可能大赚，但理智损失严重
