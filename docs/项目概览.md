**架构**: 控制台文字游戏
**最后更新**: 2026-03-28


## 三、核心数值

### 五维属性
| 属性 | 范围 | 作用 |
|------|------|------|
| INT直觉 | 10-100 | 创新、应对挑战 |
| SEN感知 | 10-100 | 操作精度、发现细节 |
| EDU知识 | 10-100 | 研究效率，理解深度 |
| STR耐力 | 10-100 | 长时间活动、继续工作 |
| SOC社交 | 10-100 | 交流、资源获取 |

### 判定系统
- 统一判定：1d100骰点
- 大成功(1) / 极难(<能力/5) / 困难(<能力/2) / 成功(<能力) / 失败(>能力且<=95) / 大失败(96-100)
- 理智修正：理智<50时×0.8，理智<30时×0.6

### 其他数值
- **理智**：0-100，归零触发异变
- **异变**：隐藏数值，影响UI，严重时死亡（≥2）
- **声望**：影响投稿成功率
- **行动点**：研一二三分别为2/3/4点
- **毕业要求**：根据导师能力1-5篇论文

## 四、科研流程

```
阅读文献(SEN判定) → 原始idea → 评估(INT判定) → 初步idea
                                              ↓
                                    实验验证(INT+SEN+EDU综合判定)
                                              ↓
                                    实验结果(理论/实验/深度三维)
                                              ↓
                        3个成熟idea → 撰写初稿 → 投稿(SOC+EDU判定)
```

## 五、导师系统

### 导师性格（压力值）
| 压力值 | 性格 | 效果 |
|--------|------|------|
| 1-30 | 佛系羊导 | 理智消耗×0.7，效率×0.8 |
| 31-60 | 中庸导师 | 无影响 |
| 61-90 | 高压导师 | 效率×1.2，理智消耗×1.3 |
| 91-100 | 无假期 | 效率×1.5，理智消耗×1.5，行动消耗×1.3 |

### 导师能力值
| 能力值 | 效果 |
|--------|------|
| <40 | 需要做横向 |
| 61-80 | 有title，毕业要求+1 |
| 81-95 | 配小导，毕业要求+2，可申请基金 |
| 96-98 | 非人类超凡生物 |
| 99 | 奈亚提亚普，需5篇顶刊 |

### 导师种族
- 人类、深潜者、米-戈、古老者、伊斯之伟大种族、奈亚提亚普
