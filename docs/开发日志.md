# 克苏鲁研究院 - 开发日志

**日期**: 2026-03-25

---

## 项目概述

**名称**: 克苏鲁研究院 (Cthulhu Academy)
**类型**: 研究生模拟 + 克苏鲁神话
**核心玩法**: 学术研究 + 理智管理 + 随机事件

---

## 已完成功能

### 基础系统
- [x] 角色系统（理智、知识、灵感、声望）
- [x] 时间系统（按周推进，研一→研三）
- [x] 行动系统（上课、研究、调查、社交、阅读、休息）
- [x] 事件系统（10+ 随机事件）
- [x] 控制台UI交互

### 数值系统
| 数值 | 作用 |
|------|------|
| 理智(SAN) | 归零则疯狂 |
| 知识 | 影响研究效率 |
| 灵感 | 触发特殊事件 |
| 声望 | 影响论文接受率 |

## 游戏阶段规划

```
研一：入门期
  - 基础课程
  - 了解神话
  - 结识NPC

研二：发展期
  - 选择研究方向
  - 建立人脉
  - 参与调查

研三：高潮期
  - 撰写毕业论文
  - 直视深渊
  - 多结局分支
```
