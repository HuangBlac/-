### 小论文测试流程
```
按 B → 跳转研二
按 2 → 阅读文献（获得原始idea）
按 E → 评估idea（输入如 1a 接受）
按 3 → 实验验证（需要≥2个实验结果变成熟）
按 4 → 撰写初稿（需要3个成熟idea）
按 5 → 投稿
```

## 2026-03-26 进展

### 今日完成
- [x] 添加4个科研方向：法术与超自然科技分析、神话文本与仪式构造、神明附属种族与独立种族、旧日支配者与外神
- [x] 添加快进时间功能（Debug模式，不改变数值只推进周数）
- [x] 添加idea评估功能（丢弃/改进/接受）
- [x] 添加研二自动抽取研究方向

### 现有不足（待解决）
1. **小论文写作模块不完整**: 原始idea转化为可实验验证的"初步idea"的逻辑未实现
2. **数值系统脱节**: 学习知识和技能树没法与科研相结合，调查模块也无法与科研发展
3. **科研方向不具体**: 目前仅是名词，缺少"主题→子方向→期刊"的完整生态
4. **理智系统单薄**: 仅作为"血量"，没有更真实的反馈
5. **随机事件逻辑问题**: 没有论文却出现"论文被引用"事件
6. **导师系统未完善**: 缺少完整的导师性格/水平系统
7. **上课与科研方向未衔接**: 选课应影响后续科研方向

### 导师系统规划（待实现）
- **导师性格** (1-100):
  - ≤30: 佛系羊导（理智消耗×0.7，科研效率×0.8）
  - 31-60: 中庸（无影响）
  - 61-90: 高压（科研效率×1.2，理智消耗×1.3）
  - ≥90: 无假期（效率×1.5，消耗×1.5，毕业要求+1，触发爆压事件）

- **导师水平** (0-99):
  - <40: 安排横向
  - >60: 有title，毕业要求+1
  - >80: 小导带（能力-20，额外学生基金）
  - >95: 非人类超凡生物
  - =99: 奈亚提亚普（必定毕业，需5篇顶刊）
