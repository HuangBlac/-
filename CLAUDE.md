# CLAUDE.md

这个文件为 Claude Code 提供"克苏鲁研究院"项目的开发指导。

## 项目概述

**克苏鲁研究院**（克研模拟器）是一个研究生模拟+克苏鲁神话题材的文字冒险游戏。

- **类型**: 文字冒险游戏 / 模拟经营
- **技术栈**: Python 3, Tkinter (GUI)
- **代码规模**: ~3500行
- **游戏主题**: 研究生科研生活 + 克苏鲁神话元素

## 核心系统

### 1. 五维属性系统
- **INT (直觉)**: 创新能力、应对挑战
- **SEN (感知)**: 操作精度、发现细节
- **EDU (知识)**: 研究效率、理解深度
- **STR (耐力)**: 长时间工作能力
- **SOC (社交)**: 交流、资源获取

### 2. 判定系统
- 基于1d100骰点
- 成功等级: 大成功(1) / 极难(<能力/5) / 困难(<能力/2) / 成功(<能力) / 失败 / 大失败(96-100)
- 理智修正: 理智<50时×0.8，理智<30时×0.6

### 3. 科研流程
```
阅读文献 → 原始idea → 评估 → 初步idea → 实验验证 → 实验结果
→ 3个成熟idea → 撰写初稿 → 投稿
```

### 4. 导师系统
- **性格**: 压力值1-100（佛系/中庸/高压/无假期）
- **能力**: 影响毕业要求（1-5篇论文）
- **种族**: 人类/深潜者/米-戈/古老者/伊斯之伟大种族/奈亚提亚普

### 5. 事件系统
- 9类事件（随机/导师/学术/神话/社交/调查/假期/娱乐等）
- 所有事件数据存储在 `game/data/events/*.json`

### 6. 异变系统
- 理智归零触发异变
- 异变等级影响UI显示和游戏结局
- 异变≥2时死亡

## 运行游戏

```bash
# 选择模式运行
python main.py

# 直接运行GUI模式
python main_gui.py
```

## 开发规范

### 代码风格
- 使用中文注释和文档字符串
- 函数命名使用snake_case
- 类命名使用PascalCase
- 保持克苏鲁神话氛围的命名和文案

### 数据管理
- 游戏配置和数据使用JSON格式存储在 `game/data/`
- 事件数据按类型分文件存储在 `game/data/events/`
- 修改数据时保持JSON格式正确

### 数值平衡
- 属性范围: 10-100
- 理智范围: 0-100
- 行动点: 研一2点/研二3点/研三4点
- 毕业要求: 根据导师能力1-5篇论文

### UI设计
- 控制台模式: 简洁的文字界面
- GUI模式: 暗色主题（#1a1a2e背景，#e94560强调色）
- 保持克苏鲁神话的诡异氛围

## 常见任务

### 添加新事件
1. 在 `game/data/events/` 对应类型的JSON文件中添加事件
2. 遵循现有事件的JSON结构
3. 测试事件触发和效果

### 调整数值平衡
1. 修改 `game/data/config.json` 中的配置
2. 或在对应系统文件中调整计算公式

### 添加新行动
1. 在 `game/action_system.py` 中定义行动
2. 在 `game/action_handlers.py` 中实现处理逻辑
3. 在 `game/game_engine.py` 中注册行动

### 修改UI
- 控制台UI: 编辑 `ui/console.py`
- 图形界面: 编辑 `ui/gui.py`

## 注意事项

- 保持游戏的克苏鲁神话氛围和黑色幽默
- 数值调整需要考虑游戏平衡性
- 新增内容应与现有系统协调
- 测试时注意边界情况（理智归零、异变触发等）
- 文案应符合研究生生活和克苏鲁神话的双重主题
