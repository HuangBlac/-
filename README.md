# 克研模拟器

研究生科研生活 + 克苏鲁神话题材的文字冒险/模拟游戏原型。

当前项目的核心价值在 `game/data/` 的课程、事件、论文题目等内容资产，以及已经搭出的课程、科研、导师、异变、毕业等系统。短期开发目标不是继续堆新功能，而是先让 Console 模式跑通第一个完整循环。

## 当前短期目标

推荐先验证 Console 最小可玩闭环：

```text
新建角色 -> 选课 -> 上课/考试 -> 解锁科研 -> 阅读文献 -> 生成 idea -> 评估 idea
```

GUI 模式仍属于后续工作；短期验收以 Console 为准。

## 运行

```bash
python main.py
```

选择 `1` 进入控制台模式。

## 测试

```bash
python -m unittest discover
```
