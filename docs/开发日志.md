# 第一部分：已完成功能（2026-03-27）

## 数值系统重做

### 1. 属性系统重做
- **新五维属性**：INT直觉、SEN感知、EDU知识、STR耐力、SOC社交
- **属性生成**：1d100随机，单项≥10，总和≥300
- **涉及文件**：`character.py`

### 2. 统一判定系统
- **判定机制**：1d100骰点
  - 1 = 大成功
  - <能力值/5 = 极难成功
  - <能力值/2 = 困难成功
  - <能力值 = 一般成功
  - >能力值 且 ≤95 = 失败
  - 96-100 = 大失败
- **理智修正**：理智<50时×0.8，理智<30时×0.6

### 3. 科研系统改造
- **阅读文献**：SEN感知判定进度，INT直觉判定创新值
- **实验验证**：INT+SEN+EDU综合判定，产生三维结果（理论/实验/深度）
- **投稿判定**：SOC+EDU综合判定
- **涉及文件**：`research.py`

### 4. 课程系统改造
- **课程属性**：INT/SEN/EDU/STR加成
- **考试判定**：1d100 < EDU + 学习次数×20
- **考试结果**：大成功满分+1d6属性，成功+1d4属性，失败需补考，大失败理智损失
- **流程**：第一学期完成后自动进入科研
- **涉及文件**：`course.py`

### 5. 异变机制
- **触发**：理智归零+1，大失败+0.05
- **效果**：
  - 每轮理智减少 = 异变值×5
  - 数字替换为符号（严重时）
  - 严重时插入乱码
  - 异变≥2死亡
- **涉及文件**：`core.py`

### 6. STR继续工作
- **触发**：STR>70时概率触发
- **效果**：不消耗本次行动点

### 7. 其他修复
- 暑假/寒假改为假期休息选项
- 课程完成后自动进入科研阶段
- 异变值作为隐藏数值

### 8. Debug功能
- **A**：快进一周
- **B**：跳转到研二
- **M**：异变+1（测试用）

## P1 - 导师系统
导师性格：1-100随机选取，越高越push，对你的要求越高，但同时给你的指导也就越多：≤30：佛系羊导，不 push，自由度高，理智消耗*0.7，但科研效率*0.8，
31–60：中庸，常规科研节奏，没有影响
61–90：高压，科研效率 ×1.2，但理智消耗 ×1.3。
≥90：无假期，无休息，行动力归零也会强行，行动力消耗*1.3，科研效率*1.5，理智消耗*1.5，毕业要求+1，但导师水平必定大于60，触发爆压事件概率增加，（爆压事件，理智回满，异常值增加，这是一个隐藏数值，只显示（*有什么东西不一样了，再也不重要了）
导师水平：0-99随机选取，越高经费越足
- <40会给你安排横向
- >60则有title,毕业要求+1
- >80则由能力值-20的小导来带你，可额外额外学生基金，毕业要求+2
- >=95非人类的超凡生物
- 99则为奈亚提亚普，必定毕业，但需5篇该研究方向顶刊才能继续跟着导师深造。
- 导师性格（1-100）：影响科研效率
- 导师水平（0-99）：影响横向/毕业要求
- 特殊导师：奈亚提亚普
其中，导师为非人类时必定研究导师所属种族，导师为奈亚子时研究课题必定为“最伟大的神明奈亚提亚普”
导师种族：深潜者，米-戈，古老者，伊斯之伟大种族（在你毕不了业的时候夺舍你），奈亚提亚普。
