# 第一部分：已完成功能（2026-03-28）

## 数值系统重做

### 1. 属性系统重做
- **新五维属性**：INT直觉、SEN感知、EDU知识、STR耐力、SOC社交
- **属性生成**：1d100随机，单项≥10，总和≥300
- **涉及文件**：`character.py`

### 2. 统一判定系统
- **判定机制**：1d100骰点
  - 1 = 大成功
  - <能力值/5 = 极难成功
  - <能力值/2 = 困难成功
  - <能力值 = 一般成功
  - >能力值 且 ≤95 = 失败
  - 96-100 = 大失败
- **理智修正**：理智<50时×0.8，理智<30时×0.6

### 3. 科研系统改造
- **阅读文献**：SEN感知判定进度，INT直觉判定创新值
- **实验验证**：INT+SEN+EDU综合判定，产生三维结果（理论/实验/深度）
- **投稿判定**：SOC+EDU综合判定
- **涉及文件**：`research.py`

### 4. 课程系统改造
- **课程属性**：INT/SEN/EDU/STR加成
- **考试判定**：1d100 < EDU + 学习次数×20
- **考试结果**：大成功满分+1d6属性，成功+1d4属性，失败需补考，大失败理智损失
- **流程**：第一学期完成后自动进入科研
- **涉及文件**：`course.py`

### 5. 异变机制
- **触发**：理智归零+1，大失败+0.05
- **效果**：
  - 每轮理智减少 = 异变值×5
  - 数字替换为符号（严重时）
  - 严重时插入乱码
  - 异变≥2死亡
- **涉及文件**：`mutation_system.py`

### 6. STR继续工作
- **触发**：STR>70时概率触发
- **效果**：不消耗本次行动点

### 7. 导师系统（新增 2026-03-28）
- **导师性格（压力值）**：
  - 1-30：佛系羊导，理智消耗×0.7，科研效率×0.8
  - 31-60：中庸导师，无影响
  - 61-90：高压导师，科研效率×1.2，理智消耗×1.3
  - 91-100：无假期导师，科研效率×1.5，理智消耗×1.5，行动消耗×1.3
- **导师能力值**：
  - <40：需要做横向
  - 61-80：有title，毕业要求+1
  - 81-95：配小导，毕业要求+2，可申请基金
  - 96-98：非人类超凡生物
  - 99：奈亚提亚普，需5篇顶刊
- **导师种族**：人类、深潜者、米-戈、古老者、伊斯之伟大种族、奈亚提亚普
- **涉及文件**：`advisor.py`

### 8. 行动系统重构（新增 2026-03-28）
- **模块化设计**：拆分为多个行动处理器
  - `CourseActionHandler` - 课程行动
  - `ResearchActionHandler` - 科研行动
  - `EntertainmentActionHandler` - 娱乐行动
  - `SocialActionHandler` - 社交行动
  - `GraduationActionHandler` - 毕业行动
- **涉及文件**：`action_system.py`、`action_handlers.py`

### 9. 假期与娱乐系统（新增 2026-03-28）
- **假期活动**：旅游、回家、娱乐、学习、休息
- **导师任务**：高压导师假期30%概率触发任务
- **娱乐活动**：听歌、玩游戏、打篮球、看电影、社交

### 10. Debug功能
- **A**：快进一周
- **B**：跳转到研二
- **M**：异变+1（测试用）

# 第二部分：项目结构

## 模块结构

| 模块 | 文件 | 功能 |
|------|------|------|
| 核心引擎 | game_engine.py | 游戏主循环、各系统协调 |
| 行动系统 | action_system.py | 行动分发、阶段管理 |
| 行动处理器 | action_handlers.py | 模块化行动执行 |
| 角色系统 | character.py | 玩家属性、时间、判定 |
| 课程系统 | course.py | 选课/上课/考试 |
| 科研系统 | research.py | idea/实验/论文/投稿 |
| 毕业系统 | graduation.py | 毕业论文 |
| 导师系统 | advisor.py | 导师分配、任务 |
| 事件系统 | event_system.py | 随机事件加载 |
| 异变系统 | mutation_system.py | 文本污染、异变效果 |
| 游戏状态 | game_state.py | 游戏结束检查 |

## 数据文件

| 文件 | 内容 |
|------|------|
| config.json | 游戏配置 |
| courses.json | 课程数据 |
| events_random.json | 随机事件 |
| events_advisor.json | 导师事件 |
| events_advisor_task.json | 导师任务 |
| events_academic.json | 学术事件 |
| events_mythos.json | 神话事件 |
| events_social.json | 社交事件 |
| events_investigation.json | 调查事件 |
| events_holiday.json | 假期事件 |
| events_entertainment.json | 娱乐活动 |

## 核心数值

- **理智**：0-100，归零触发异变
- **五维属性**：INT/SEN/EDU/STR/SOC，10-100
- **异变**：隐藏数值，影响UI

## 游戏流程

```
研一上 → 选课 → 上课 → 期末考试 → 研一假期
    ↓
研二/研三 → 阅读文献 → 获得Idea → 评估Idea → 实验 → 写论文 → 投稿
    ↓
研三 → 毕业论文 → 盲审 → 答辩 → 毕业
```
