# 克苏鲁研究院 - 项目概览

**项目类型**: 研究生模拟 + 克苏鲁神话文字冒险游戏
**代码规模**: ~1750行 Python
**架构**: 控制台文字游戏

---

## 一、游戏流程

```
研一 → 研二 → 研三
  ↓      ↓      ↓
课程学习 科研    毕业
  ↓      ↓      ↓
考试    发论文   答辩/疯狂
```

---

## 二、模块结构

### 2.1 核心模块 (game/core.py)
- **作用**: 游戏主循环、行动调度、状态管理
- **关键类**: `GameCore`
- **功能**:
  - 行动分发（上课/研究/调查/社交/休息）
  - 时间推进（按周）
  - 随机事件触发
  - 游戏结束判定

### 2.2 角色模块 (game/character.py)
- **作用**: 玩家与NPC数据
- **关键类**: `Player`, `NPC`, `SemesterType`, `ResearchDirection`
- **功能**:
  - 玩家属性（理智、知识、灵感、声望）
  - 时间系统（年级、学期、周数）
  - 技能树
  - 人际关系

### 2.3 科研系统 (game/research.py)
- **作用**: 论文写作全流程
- **关键类**: `ResearchSystem`, `Idea`, `ExperimentResult`, `Paper`
- **功能**:
  - 阅读文献 → 获取原始idea
  - idea评估（接受/丢弃/改进）
  - 实验验证 → 产生实验结果
  - 撰写初稿
  - 投稿与审稿

### 2.4 课程系统 (game/course.py)
- **作用**: 研一课程学习
- **关键类**: `Course`, `CourseSystem`, `ExamSystem`
- **功能**:
  - 选课（必修+选修）
  - 课程技能加成
  - 期末考试
  - 研究方向决定

### 2.5 事件系统 (game/event_system.py)
- **作用**: 随机事件管理
- **关键类**: `EventSystem`
- **功能**:
  - 随机事件触发
  - 事件条件判定
  - 事件效果应用

### 2.6 毕业系统 (game/graduation.py)
- **作用**: 研三毕业论文
- **关键类**: `GraduationThesis`
- **功能**:
  - 开题报告
  - 论文撰写
  - 答辩

### 2.7 UI模块 (ui/console.py)
- **作用**: 控制台界面
- **关键类**: `ConsoleUI`
- **功能**:
  - 菜单显示
  - 输入获取
  - 状态展示

## 四、关键数值

### 玩家属性
| 属性 | 范围 | 作用 |
|------|------|------|
| 理智(SAN) | 0-100 | 归零则疯狂 |
| 知识 | 0-∞ | 研究效率 |
| 灵感 | 0-∞ | 创新加成 |
| 声望 | 0-∞ | 投稿成功率 |

### 时间
- 研一：上8周 + 暑4周 + 下9周 + 寒2周 = 23周
- 研二/研三：同上
- 每周行动点：研一2点，研二3点，研三4点
