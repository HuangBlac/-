# 克研模拟器

研究生科研生活 + 克苏鲁神话题材的文字冒险/模拟游戏原型。

## 项目介绍

你说得对，但是“克研模拟器”是由BlacHuang自主研发的一款模拟经营类游戏，游戏中，你将扮演一名叫{user}的神秘角色，前往克苏鲁研究院，进行从研一新生上课，到研二阅读论文，接触丰富多彩的实验课题，进行多种形式的实验，最终发表论文，甚至毕业

但正如提瓦特的虚假天空，面对邪神的研究注定是一场艰苦的旅程。你需要平衡学业压力，经受住面对未知带来的异变的恐惧，直面深渊？或者，其实我们并不能做到这一点？

## 
```text
新建角色 -> 选课 -> 上课/考试 -> 解锁科研 -> 阅读文献 -> 生成 idea -> 评估 idea -> 进行实验 -> 撰写论文 -> 获得论文 -> 毕业论文 -> 盲审二审 -> 答辩 -> 毕业
```

## 运行
首先clone到本地一个你喜欢的文件夹
```bash
git clone 
```
然后命令行里面运行这个命令
```bash
python main.py
```

选择 `1` 进入控制台模式，`2`暂时还不能用

然后就可以开始你的研究生生活了！
